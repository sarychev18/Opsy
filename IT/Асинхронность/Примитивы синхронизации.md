|               |                                                   |                                |
| ------------- | ------------------------------------------------- | ------------------------------ |
| **Lock**      | Блокировка доступа к ресурсу (1 поток за раз)     | `with lock:`                   |
| **Semaphore** | Ограничение количества потоков (N потоков за раз) | `Semaphore(5)`                 |
| **Event**     | Сигнал между потоками (wait/trigger)              | `event.wait()`, `event.set()`  |
| **Condition** | Условная блокировка (wait/notify)                 | `cond.wait()`, `cond.notify()` |
| **Queue**     | Потокобезопасная очередь                          | `queue.put()`, `queue.get()`   |

```python
# ✅ Lock пример
lock = threading.Lock()
def safe_increment():
    global counter
    with lock:  # ✅ Только 1 поток внутри
        counter += 1

# ✅ Semaphore пример
semaphore = threading.Semaphore(3)  # ✅ Максимум 3 потока
def limited_task():
    with semaphore:
        process()

# ✅ Event пример
event = threading.Event()
def waiter():
    event.wait()  # ⏳ Ждёт сигнала
    print("Event triggered!")
def trigger():
    event.set()  # ✅ Сигнал всем
```
