#flashcards/FastAPI 
Path-параметры — переменные в URL-пути (/users/{id}). Обязательны по умолчанию, т.к. без них маршрут не существует. FastAPI автоматически парсит их в указанный тип (int, str). Дополнительная валидация — через класс Path() (лимиты, regex). Отличаются от Query: Path идентифицирует ресурс, Query — опции запроса.
<!--SR:!2025-12-19,1,210-->

**Важно**
- ✅ **Порядок важен:** endpoint'ы с path-параметрами должны быть **после** фиксированных путей
- ✅ Endpoint - конечная точка API

**Подтермины**
- "Endpoint" - конечная точка API

```python
# config.py
@app.get("/users/{user_id}")          # {user_id} в пути
def get_user(user_id: int):           # Аннотация типа обязательна
    return {"id": user_id}
```
