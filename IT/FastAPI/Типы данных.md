#flashcards/FastAPI 
Можно ли писать свои типы данных?  :: Да, через Pydantic модели. Создаем класс, наследуемый от BaseModel, объявляем поля с аннотациями типов. FastAPI использует их для: валидации, сериализации, документации. </br>Также можно и без Pydantic, просто реализовать класс и функциях в ручках проверять входные данные
<!--SR:!2026-05-11,56,210-->

**Важно**
- ✅ Также можно и без Pydantic, просто реализовать класс и функциях в ручках проверять входные данные 

```python
from pydantic import BaseModel

class Item(BaseModel):          # Свой тип данных
    name: str                   # Обязательное поле
    price: float                # Обязательное
    tax: float | None = None    # Опциональное

@app.post("/items/")
def create_item(item: Item):    # FastAPI валидирует автоматически
    return item
```
Уровень знания данной темы: ознакомлен

#flashcards/FastAPI 
в чем отличие между redoc и doc?  :: FastAPI автоматически генерирует OpenAPI-спецификацию (JSON). /docs (Swagger UI) и /redoc — это два разных интерфейса для визуализации этой спецификации. Swagger — интерактивный (можно тестировать запросы), Redoc — более красивый, но без интерактива.
<!--SR:!2026-05-03,48,230-->

Нюансы
- ✅ Swagger (/docs) — для разработки и тестирования
- ✅ Redoc (/redoc) — для документации команде/заказчику
