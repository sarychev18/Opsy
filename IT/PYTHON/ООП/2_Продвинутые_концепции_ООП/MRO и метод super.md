**MRO (Method Resolution Order)** - это порядок разрешения методов в Python при множественном наследовании. Он определяет, в каком порядке интерпретатор будет искать метод в иерархии классов. В Python 3 используется алгоритм C3 Linearization, который гарантирует сохранение порядка в родительских классах и монотонность. Порядок MRO можно посмотреть через атрибут `Class.__mro__` или метод `Class.mro()`._
```python
class A:
    def method(self):
        return "A"

class B(A):
    def method(self):
        return "B"

class C(A):
    def method(self):
        return "C"

class D(B, C):
    pass

# Смотрим порядок MRO
print(D.__mro__)
# (<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>)

d = D()
print(d.method())  # "B" - ищет в порядке D -> B -> C -> A
```

### Алгоритм C3 Linearization

Python 3 использует алгоритм C3, который следует трем правилам:

1. **Предок перед потомком** - родительские классы идут после дочерних
2. **Сохранение порядка** - порядок родительских классов сохраняется
3. **Монотонность** - если класс A предшествует классу B во всех случаях, то так будет и в MRO

## Diamond problem
Проблема ромба - Это ситуация, когда класс наследуется от двух классов, которые в свою очередь наследуются от одного общего базового класса. 
```python
class A:
    def say_hello(self):
        print("Привет из A")


class B(A):
    pass


class C(A):
    def say_hello(self):
        print("Привет из C")


class D(B, C):
    pass


obj = D()
obj.say_hello() # Привет из C

print(D.__mro__) # (<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>)
```
Обрати внимание, что после B идёт класс С, а не А

Проблема ромба решается автоматически с помощью MRO.

## Super()
Super() - это функция, которая обращается к методам родительского класса. Однако она идёт не напрямую к родительскому классу, а к следующему по порядку MRO
```python
class A:
    def say_hello(self):
        print("Привет из A")


class B(A):
    def say_hello(self):
        super().say_hello()


class C(B):
    pass


obj = C()
obj.say_hello() # Привет из A
```
