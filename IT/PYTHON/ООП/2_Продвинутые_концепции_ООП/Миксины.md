Что такое Микисны :: Микисны - это классы, которые предоставляют дополнительную реализацию(методы) с помощью множественного наследования. При создании класса миксин, нужно писать в конце имени mixin (пример: class SerializableMixin:), они не предназначены для собственного использования, а для наследования.
