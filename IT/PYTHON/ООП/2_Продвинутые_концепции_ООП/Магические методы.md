Магические методы - это стандартные методы в пайтон. 
Допустим мы хотим посчитать длину нашего объекта
```python
class Collection:
    def __init__(self, list):
        self.list = list

collection = Collection(list)
len(collection)

#Выдаст ошибку → TypeError: object of type 'Collection' has no len()
# пайтон не умеет считать объекты наших классов
```
Чтобы он умел обрабатывать объекты наших классов, нужно предусматривать это
```python
class Collection:
    def __init__(self, list):
        self.list = list

    def __len__(self):
        return len(self.list)

collection = Collection([1, 2, 3])
print(len(collection)) # 3
```
Теперь пайтон умеет считать наши объекты

Магические методы записываются с нижним подчеркиванием с обеих сторон.
```python
    def __len__(self):
```

# Callable-объекты
Callable-объект - это объект, который можно вызвать с помощью круглых скобок, возможно с аргументами. Проверить является ли объект Callable-объектом с помощью функции callable()

Примеры:
#### 1. Функции

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Hello, Alice!
print(callable(greet)) # True
```

#### 2. Классы (при создании экземпляров)
```python
class Person:
    def __init__(self, name):
        self.name = name

# Класс является callable - создает экземпляр
person = Person("Bob")
print(callable(Person))  # True
```

## Важные особенности

1. **Все callable-объекты имеют метод `__call__`**
2. **Проверка через `callable()` более безопасна, чем попытка вызова**
3. **Callable-объекты могут иметь состояние (в отличие от чистых функций)**
4. **Широко используются в паттернах проектирования и фреймворках**
