Метаклассы нужны для того, чтобы создать классы или изменить поведение других классов. 

Пример - создадим класс, который делает названия методов написанными большими буквами. 
```python
class UpperCaseMeta(type):
    """Метакласс для переименовывания методов в верхний регистр"""
    def __new__(cls, name, bases, dct):
        new_dct = {}
        for key, value in dct.items():
            if not key.startswith('__'):
                new_dct[key.upper()] = value
            else:
                new_dct[key] = value
        return super().__new__(cls, name, bases, new_dct)


class MyClass(metaclass=UpperCaseMeta):
    def method(self):
        ...

    def another_method(self):
        ...


instance = MyClass()
instance.METHOD()  # Методы теперь в верхнем регистре
instance.ANOTHER_METHOD()
```
Можно использовать атрибут `__metaclass__` используется для обозначения метакласса, который необходимо использовать для создания класса.
```python
class MyClass:
    __metaclass__ = UpperCaseMeta
```

Что такое "метаклассы" в Python? :: Метаклассы в Python — это классы, которые управляют поведением других классов. Они определяют, как должны создаваться и функционировать классы, и позволяют изменять их поведение во время создания, например, добавлять методы или атрибуты.

По умолчанию, метаклассом для всех классов является type. Метаклассы полезны для реализации сложных паттернов проектирования, таких как Singleton, или для автоматической регистрации классов.
