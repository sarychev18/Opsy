Что такое `__slots__` и зачем они нужны?  ::  __slots__ — это атрибут, который позволяет ограничить набор атрибутов для экземпляра класса, что улучшает производительность и уменьшает потребление памяти.
<!--SR:!2026-04-10,18,139-->

class Point:
    __slots__ = ('x', 'y')

    def __init__(self, x, y):
        self.x = x
        self.y = y

p = Point(1, 2)
print(p.x)  # Вывод: 1
print(p.y)  # Вывод: 2

Как обозначить свойства класса?  :: @property
<!--SR:!2026-11-12,221,219-->

class Circle:
    def __init__(self, radius):
        self._radius = radius

    @property
    def radius(self):
        return self._radius

    @radius.setter
    def radius(self, value):
        if value < 0:
            raise ValueError("Радиус не может быть отрицательным")
        self._radius = value

"@property — это декоратор, который позволяет обращаться к методу как к атрибуту. полезно для инкапсуляции и контроля доступа к атрибутам, а также для обеспечения безопасности и целостности данных."
