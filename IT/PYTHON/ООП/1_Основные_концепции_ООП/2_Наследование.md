Это когда дочерний класс наследует методы и свойства родительского класса. Позволяет переиспользовать код 

```python
# Родительский класс
class Animal:
    def make_sound(self):
        print("Издаёт какой-то звук")

# Дочерний класс
class Dog(Animal):
    pass  # Не добавляем новых методов (пока)

# Использование
dog = Dog()
dog.make_sound()  # Вывод: Издаёт какой-то звук
```
Тут мы видим, что class Dog(Animal) → унаследовал все что было у класса Animal

Что такое наследование в ООП ? :: Это когда дочерний метод наследует все методы и свойства родительского класса.
