Полиморфизм - переводится как "много форм"
Это возможность обработать объекты из разных классов, использую один и тот же метод или переопределяя его
```python
# Родительский класс
class Animal:
    def eat(self):
        print("Mmm, tasty")

    def make_sound(self):
        print("Some generic sound")

class Cat(Animal):
    def make_sound(self):
        print("Meow")

class Dog(Animal):
    def make_sound(self):
        print("Woof")

# Использование
cat = Cat()
cat.make_sound() # Вывод: Meow - переопределили для нашего класса
cat.eat() # Вывод: Mmm, tasty - использовали из класса-родителя

dog = Dog()
dog.make_sound()  # Вывод: Woof
```
Тут можно увидеть, что метод называется одинаково и его переопределили.
