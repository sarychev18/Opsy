### Сериализация данных

Сериализация данных - это процесс преобразования данных в байткод хранением данных (pickle) или для преобразования данных в Json для работы с файлами. Десериализация - обратный процесс. 

**Основные моменты**

- Основные модули: `pickle` для бинарной сериализации Python-объектов, `json` для текстовой сериализации

- `pickle` работает с любыми объектами Python, но небезопасен для ненадежных данных

- `json` безопасен, но работает только с базовыми типами данных

- Протоколы pickle: от 0 (текстовый) до 5 (бинарный, Python 3.8+)

- Для сложных объектов нужно реализовывать методы `__getstate__` и `__setstate__`

```python
# 1. Базовый пример с pickle
import pickle

# Данные для сериализации
data = {
    'name': 'Анна',
    'age': 30,
    'pets': ['кот', 'собака'],
    'scores': (95, 87, 92)
}

# Сериализация в bytes
serialized_data = pickle.dumps(data)
print(f"Сериализованные данные (первые 50 байт): {serialized_data[:50]}...")

# Десериализация
deserialized_data = pickle.loads(serialized_data)
print(f"Десериализованные данные: {deserialized_data}")

# Сохранение в файл
with open('data.pkl', 'wb') as f:  # 'wb' - write binary
    pickle.dump(data, f)

# Загрузка из файла
with open('data.pkl', 'rb') as f:  # 'rb' - read binary
    loaded_data = pickle.load(f)
    print(f"Данные из файла: {loaded_data}")
    
# 2. Работа с JSON
import json

data = {
    'name': 'Борис',
    'age': 25,
    'active': True,
    'hobbies': ['чтение', 'программирование']
}

# Сериализация в JSON строку
json_string = json.dumps(data, indent=2, ensure_ascii=False)
print("JSON строка:")
print(json_string)

# Десериализация из JSON строки
parsed_data = json.loads(json_string)
print(f"Имя: {parsed_data['name']}")

# Работа с файлами
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

with open('data.json', 'r', encoding='utf-8') as f:
    file_data = json.load(f)
    print(f"Данные из JSON файла: {file_data}")
```
