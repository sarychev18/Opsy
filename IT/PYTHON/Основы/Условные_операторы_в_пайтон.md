Тернарный оператор (также известный как условное выражение) — это синтаксическая конструкция, которая позволяет компактно записать простое условие выбора между двумя значениями. Он называется "тернарным" потому что принимает три операнда: условие, значение для истинного случая и значение для ложного случая.

**Основные характеристики тернарного оператора:**

- **Компактность:** заменяет многострочный if-else одной строкой
    
- **Функциональность:** всегда возвращает значение (выражение, а не statement)
    
- **Ленивость:** вычисляется только одна ветвь (short-circuit evaluation)
    
- **Читаемость:** улучшает для простых условий, но может ухудшать для сложных
```python
age = 18
status = "Взрослый" if age >= 18 else "Ребёнок"
print(status)  # Выведет "Взрослый"
```

Что такое match/case? :: это новый способ писать условия вместо if/elif/else. Добавлен в Python 3.10
<!--SR:!2026-06-11,72,180-->

```python
# Старый подход с if/elif/else
def old_style(status):
    if status == 200:
        return "OK"
    elif status == 404:
        return "Not Found"
    elif status == 500:
        return "Internal Server Error"
    else:
        return "Unknown status"

# Новый подход с match/case
def new_style(status):
    match status:
        case 200: return "OK"
        case 404: return "Not Found"
        case 500: return "Internal Server Error"
        case _: return "Unknown status"

def handle_order_status(status):
    match status:
        case "pending":
            print("Заказ ожидает обработки")
        case "shipped":
            print("Заказ в пути")
        case "delivered":
            print("Заказ доставлен")
        case _:
            print("Неизвестный статус")

# Использование
handle_order_status("shipped")  # Вывод: "Заказ в пути"
```
