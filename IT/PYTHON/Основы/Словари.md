Словарь - это изменяемый, упорядоченный тип данных.
Имеет ключ и значение.
Название словаря = {'Ключ' : 'Значение')

Ключ - должен быть из неизменяемым типом данных (str, tuple)
Ключ может быть изменяемым типом данных, также может состоять и из словаря.
Можно создать 2 способами
country = {'Russia' : '+7') или
country = dict(Russia='+7')

Основные методы:

.items() - возращает ключ и значение
.keys() - возращает ключ
.values() - возращает значение
.pop() - удаляет ключ
.get() - возвращает None или другое слово - .get(keys, ключ - если нет в словаре)
.clear() - удаляет содержимое словаря

Словари используют хеш-функцию для х**р**анения данных.
Они создают таблицу либо с пустым значением, либо с Хэш данными (когда есть ключ - значение)

В этой таблице есть ячейки, иногда хэш функции дают одинаковые индексы для разных ключей (коллизия)

В пайтон используется для таких случаев комбинацию открытой адресации. (Означает, что при столкновении выполняется поиск свободных слотов, если есть перезаписывается, если нет, то таблица расширяется)

Среднее время работы с элементом в словаре составляет O(1) (BIG O - константное время) - НЕ зависит от объема данных. Самое минимальное возможное время.
Python периодически оптимизирует таблицу словарей (перехеширование) для оптимальной работы.

С версии 3,6 в пайтон используется упорядоченное хранение добавленных ключей - т.е. до это версии словари считались неупорядоченным типом данных
