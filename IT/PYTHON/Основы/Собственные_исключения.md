Собственные исключения создаются самостоятельно разработчиком, если стандартных типов недостаточно

```python
class NegativeNumberError(Exception): #Создается исключение со своим названием
    pass #Ставится заглушка

def calculate_square_root(number):
    if number < 0:
        raise NegativeNumberError("Нельзя вычислить квадратный корень отрицательного числа")
    return number ** 0.5

print(calculate_square_root(-1))  # Возбудит исключение NegativeNumberError
```
raise используется для принудительного возбуждения исключения
pass ставиться, если не планируется дополнительно изменять метод исключения
