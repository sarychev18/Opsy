Как можно скачать конкретную версию библиотеки через pip?  :: **`pip install package==version`**. Можно указывать точную версию, диапазоны версий или совместимые версии. Это критически важно для воспроизводимости окружения и стабильности проекта.
<!--SR:!2026-04-15,113,250-->

```python
# Операторы сравнения версий
pip install package==1.4.2      # точно 1.4.2
pip install package>=1.4.0      # 1.4.0 или выше
pip install package<=2.0.0      # 2.0.0 или ниже
pip install package>1.3.0       # строго больше 1.3.0
pip install package<2.0.0       # строго меньше 2.0.0

# Диапазоны версий
pip install "package>=1.0.0,<2.0.0"     # от 1.0.0 до 2.0.0 (не включая)
pip install "package>1.0.0,!=1.5.0"     # больше 1.0.0, но не 1.5.0

# Совместимые версии (compatible release)
pip install package~=1.4        # >=1.4.0, ==1.*
pip install package~=1.4.2      # >=1.4.2, ==1.4.*
```
