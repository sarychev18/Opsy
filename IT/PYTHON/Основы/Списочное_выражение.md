Это компактный способ создать список (list comprehension)
Вместо 
```python
for i in range(10):
    if i < 5:
        print(i)
    else:
        print('не подходит')
```
Вот такой код:
```python
[print(i) if i < 5 else print('не подходит') for i in range(10)]
```
