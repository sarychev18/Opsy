Замыкание - функция, которая "запоминает" переменную из внешней среды, даже если эта функция уже завершила свой жизненный цикл.

Пример.
```python
def adder(base_number):
    def add(x):
        return base_number + x
    return add

adder_five = adder(5)
print(adder_five(3))  # Результат: 8
print(adder_five(2))  # Результат: 7
```
Тут первом вызове adder(5) - base_number получает значение 5 и запоминает как 'X'
А уже при вызове adder_five(3) - base_number получает значение 3 и прибавляет 5(Х)
и получаем 8
Обрати внимание, что Х теперь будет всегда 5
