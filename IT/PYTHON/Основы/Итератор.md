Итератор  - это объект, который позволяет двигаться по последовательности элементов (список, словарь, кортеж, множество, текст). чтобы считаться итератором - объект должен иметь 2 функции:
```python
__iter__() - возвращает сам объект для работы с циклом FOR
__next__() - двигается к следующему элементы, если элементов нет выволит сообщение StopIteration
```

Пример итерирования по словарю:
```python
my_dict = {"apple": 1, "banana": 2, "orange": 3}

for fruit in my_dict.keys():
    print(fruit)

Вывод:
apple
banana
orange
```

Является ли Range итератором? :: Range НЕ является итератором, но является итерируемым объектом. У range есть метод `__iter__()`, но нет метода `__next__()`.
