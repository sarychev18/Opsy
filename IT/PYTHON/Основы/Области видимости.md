Расскажи что-нибудь про области видимости ?  :: **Локальная область видимости (local scope)** — это пространство имён, созданное **внутри функции**. Переменные, объявленные здесь, видны **только внутри этой функции** и исчезают после её завершения.<br>**Глобальная область видимости (global scope)** — это пространство имён на уровне **модуля** (файла .py). Переменные, объявленные здесь, видны **во всём модуле**, включая код внутри функций.
<!--SR:!2026-08-06,125,250-->

**Важно**
- ✅ **Чтение:** Внутри функции можно **читать** значение глобальной переменной.
- ✅ **Запись/Изменение:** Чтобы **изменить** глобальную переменную внутри функции, нужно использовать ключевое слово **`global`**.
- ✅ **Вложенные функции:** Чтобы изменить переменную из внешней (неглобальной) функции во внутренней, нужно использовать ключевое слово **`nonlocal`**.

```python
# ГЛОБАЛЬНАЯ область видимости
global_var = "Я глобальная"

def outer_function():
    # ЛОКАЛЬНАЯ для outer_function область видимости
    outer_var = "Я из outer"
    
    def inner_function():
        # ЛОКАЛЬНАЯ для inner_function область видимости
        local_var = "Я строго локальная"
        
        # 1. Чтение глобальной переменной - РАБОТАЕТ
        print(global_var)  # "Я глобальная"
        
        # 2. Чтение переменной из внешней функции - РАБОТАЕТ (замыкание)
        print(outer_var)   # "Я из outer"
        
        # 3. Попытка ИЗМЕНИТЬ глобальную переменную (БЕЗ global) - СОЗДАСТ локальную!
        global_var = "Попытка изменить"  # Это новая локальная переменная inner_function!
    
    inner_function()
    print("После inner:", global_var)  # Останется "Я глобальная"

outer_function()

# 4. Правильное изменение глобальной переменной
def correct_global_change():
    global global_var  # Явное объявление
    global_var = "Изменено через global"
    print("Внутри функции:", global_var)

correct_global_change()
print("Снаружи:", global_var)  # "Изменено через global"

# 5. Использование nonlocal
def outer():
    counter = 0
    def inner():
        nonlocal counter  # Указываем, что хотим работать с переменной из внешней функции
        counter += 1
        return counter
    return inner

func = outer()
print(func())  # 1
print(func())  # 2
```
