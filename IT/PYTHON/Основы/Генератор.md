Генераторы позволяют создавать итераторы без необходимости писать классы с методами `__iter__()` и `__next__()`. Основная идея генераторов — "ленивые вычисления", когда значения производятся по мере необходимости, а не все сразу хранятся в памяти.

Плюсы: 
• позволяет экономить память, так как производит вычисления по одному (т.е. не хранит информацию о других элементах)

Генератор использует ключевое слово `yield` вместо `return` , что позволяет хранить лишь один элемент в памяти. а не всю последовательность, которая могла бы быть при `return`

Если бы использовали `return` → Вернулось бы только первое значение, остальное игнор

**Основные характеристики генераторов:**

- **Ленивость:** значения вычисляются только при обращении

- **Экономия памяти:** не хранят всю последовательность в памяти

- **Состояние:** сохраняют состояние выполнения между вызовами

- **Одноразовость:** после полного прохода нельзя использовать повторно

- **Управление потоком:** могут быть бесконечными или ограниченными

```python
def normal_function():
    return 1
    return 2
    return 3
```
Благодаря `yield` вернуться все значения.
```python
def normal_function():
    yield 1
    yield 2
    yield 3
```
Чтобы вернуть все значения нужно использовать функцию `next()`
`next()` - это ручной способ передвигать генератор на следующее значение
```python
def my_generator():
    yield 1
    yield 2
    yield 3

gen = my_generator()
print(next(gen))  # получаем первый элемент, 1
print(next(gen))  # второй элемент, 2
```
Лучше использовать цикл FOR
```python
for item in my_generator():
    print(item)  # печатает все элементы одного за другим
```

### **Генераторные выражения (generator expressions)**
Еще есть возможность кратко записывать генераторы, без **yield**
```python
squares = (x*x for x in range(10))

for square in squares:
    print(square)  # вывод квадратов чисел от 0 до 9
```
тоже самое что и генераторная функция
```python
def generate_squares(n):
    for x in range(n):
        yield x * x

squares = generate_squares(10)

for square in squares:
    print(square)
```
