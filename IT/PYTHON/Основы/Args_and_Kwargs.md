```python
*args - 1 знак *
```
\*args - (agrumens - аргументы) используется для передачи неограниченного кол-ва НЕименованных аргументов.

Пример:
```python
def my_funcion(*names):
    for name in names:
        print(name)
#Вывод
Анна
Данна Уайт
Александр Волков
Тамби
```
Создаётся кортеж \*args

```python
**kwargs - 2 знака **
```
\*\*kwargs - (keywords agrumens - именные аргументы) используется для передачи неограниченного кол-ва именованных аргументов.

Пример:
```python
def my_funcion(**names):
    for key, value in names.items():
        print(f"{key} : {value}")

my_funcion(name1='Анна', name2='Данна Уайт', name3='Александр Волков', name4='Тамби')
#Вывод
Name1 : Анна
Name2 : Данна Уайт
Name3 : Александр Волков
Name4 : Тамби
```
Создаётся словарь \*\*kwargs
