Что такое O(n)? :: O(n) - это обозначение линейной сложности алгоритма, где время расчет пропорционально количеству входных данных. О - нотация используется для анализа сложности алгоритма и прогнозирования сложности при нагрзуке

```python
# O(n) - линейная сложность
def process_data(data):
    for item in data:  # n операций
        process(item)   # время ~ n

# O(1) - постоянная сложность  
def get_first(data):
    return data[0]     # всегда 1 операция

# O(n²) - квадратичная сложность
def find_pairs(data):
    for i in data:     # n раз
        for j in data: # n раз
            check(i, j) # время ~ n²
```
## Основные O-сложности операций

### **По возрастанию сложности:**

1. **O(1)** - постоянная (доступ по индексу)
2. **O(log n)** - логарифмическая (бинарный поиск)
3. **O(n)** - линейная (линейный поиск)
4. **O(n log n)** - линейно-логарифмическая (быстрая сортировка)
5. **O(n²)** - квадратичная (пузырьковая сортировка)
6. **O(2^n)** - экспоненциальная (рекурсивные Фибоначчи)
7. **O(n!)** - факториальная (задача коммивояжера)
