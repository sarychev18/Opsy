Это как бы подъязык. 
Используются для работы с текстом. Позволяют:
• Искать соответствия,
• Изменять,
• Извлекать.

Регулярными выражениями можно пользоваться с любым языком. 
В Python чтобы пользоваться регулярными выражениями (Regular expressions→ Шаблонные выражения) нужно импортировать модуль re (**import re**)

**Основные концепции регулярных выражений:**
- `\d` - цифры, `\w` - буквы/цифры, `\s` - пробелы
- `*` - 0 или более, `+` - 1 или более, `?` - 0 или 1
- `{n}` - точно n раз, `{n,m}` - от n до m раз
- `[abc]` - любой из символов, `[^abc]` - любой кроме

Чтобы искать нужные символы в тексте используется свои символы (**Метасимволы**)

| Символ | Значение                                                                |
| ------ | ----------------------------------------------------------------------- |
| .      | Любой одиночный символ                                                  |
| ^      | Начало строки                                                           |
| $      | Конец строки                                                            |
| *      | Ноль или более повторений предыдущего символа или группы                |
| +      | Один или более повторений предыдущего символа или группы                |
| ?      | Ноль или одно повторение предыдущего символа или группы                 |
| {}     | Определенное число повторений, диапазон повторений                      |
| []     | Класс символов (например, `[abc]` совпадает с одним символом из набора) |
| ()     | Группа символов, позволяющая сохранить найденное значение               |
| \|     | Альтернатива (логическое OR), выбирает между двумя вариантами           |
| \d     | Цифра ([0-9])                                                           |
| \D     | Не цифра (все, кроме цифр)                                              |
| \w     | Словесный символ (буква, цифра или подчёркивание)                       |
| \W     | Не словесный символ                                                     |
| \s     | Пробел (пробел, табуляция, новая строка)                                |
| \S     | Непробельный символ                                                     |
| \b     | Граница слова (перед началом или концом слова)                          |
| \B     | Внутри слова (не граница слова)                                         |
| \\     | Экранирование специального символа                                      |

### Методы

Метод `.search()` ищет первое совпадение в строке:

Метод `.match()` проверяет наличие соответствия в самом начале строки:

Метод `.findall()` возвращает список всех найденных совпадений:

Метод `.sub()` позволяет заменить части строки, соответствующие регулярному выражению:


```python
import re

# Основные методы модуля re:
text = "Мои номера: 123-456-7890 и 555-1234"

# 1. search - поиск первого совпадения
match = re.search(r'\d{3}-\d{3}-\d{4}', text)
print(match.group())  # 123-456-7890

# 2. findall - поиск всех совпадений
numbers = re.findall(r'\d{3}-\d{4}', text)  # ['555-1234']

# 3. sub - замена по шаблону
clean_text = re.sub(r'\d', 'X', text)  # "Мои номера: XXX-XXX-XXXX и XXX-XXXX"

# 4. match - проверка с начала строки
is_valid = re.match(r'^\w+', text)  # True

# 5. split - разделение по шаблону
parts = re.split(r': ', text)  # ['Мои номера', '123-456-7890 и 555-1234']
```
